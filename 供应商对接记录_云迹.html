供应商对接记录 · 云迹科技（UP送物 + 咖啡机器人）

对接日期：2026年7月 · 对接方：云迹科技 · 文档来源：云迹开放平台API文档

一、云迹UP机器人（送物/巡游）

1.1 API概览

项目 内容
平台 云迹开放平台 docs.yunjiai.cn
API版本 v3-beta
接口基础 POST /v3/stores/{store-id}/robot/task
认证 需store-id（门店ID）+ Token
任务类型 3种：移动、送物、巡游

1.2 三种任务类型

类型 action参数 说明 回充
移动任务 move-task 前往点位停留，不回充 ❌
送物任务 transport 途径点取物→送目标→自动回充 ✅
巡游任务 cruise 循环宣传（仅润系列） ❌

1.3 送物任务核心参数

参数 说明
robotId 机器人ID
cabinId 舱体ID（分体机器人）
autoBack 送物后自动回充
target[].point 目标点位
target[].lockers 锁柜（密码+二维码取物）
target[].notification 到达通知（VMS/短信）
target[].via 途径点（先取物再送）

二、咖啡机器人（HTCBot开放平台）

2.1 平台概览

项目 内容
厂商 HTCBot
API地址 https://coffee-test.htcbot.com
认证 双重认证（MD5签名+Bearer Token）
Token有效期 access_token 2h / refresh_token 7天

2.2 核心API

接口 地址 用途
4.1 获取令牌 /gwoauth/v1/user/auth/token appId+appSecret换token
4.4 订单查询 /gwoauth/v1/order/detail/queryPageByOpen 查订单状态
4.5 订单制作 /gwoauth/v1/order/detail/createOpenOrder 创建订单+拉花
4.8 菜单查询 菜单接口 获取设备商品列表
4.11 印花图库 图库接口 花式印花选图
4.12 上传图片 上传接口 AI印花自定义上传

2.3 拉花支持（菜单type分类）

type 说明 图片来源
0 普通菜单 无拉花
1 花式印花 4.11图库选图
2 AI印花 4.12上传自定义图片

2.4 人脸拉花完整链路

拍照（iPad/品牌墙摄像头）
  → 中控StyleGAN生成线条画（<2秒）
  → 上传图片（4.12 API）获取URL
  → 创建订单（4.5 API，flower=URL，type=2）
  → 咖啡机制作+拉花（90秒）
  → 取餐屏人脸识别匹配取餐

四、对场景的影响

场景4 送物接力 — 完美匹配

云迹transport API直接支持：途径点取物→走廊导航→到达通知→二维码取物→自动回充。DAG引擎调REST API即可，无需额外硬件开发。

场景2 拍照接力→咖啡 — 技术链路确认

HTCBot AI印花API（type=2+4.12上传+4.5.flower）完美匹配人脸拉花。StyleGAN生成→上传→下单传flower URL。

场景6 充电调度 — 简化

UP自带autoBack自动回充，无需额外调度。充电调度系统排除UP。
