一、核心问题

1.1 当前方案的本质

当前联动方案是"触发→响应"的if-then规则，即使使用DAG编排，本质仍是预编程的流水线：

当前：固定脚本式
  访客签到 → Walker迎宾 → M20表演 → 咖啡制作 → 送包

问题：
  - 每次执行相同步骤，缺乏灵活性
  - 任何环节出错就全链路中断
  - 机器人不会"思考"，只会"执行"
  - 无法应对未预料的情况

1.2 固定响应 vs 主动式智能

层级 固定响应（当前） 主动式智能（目标）
感知 RFID触发/按钮 多模态持续感知（视觉+听觉+环境）
决策 预编程DAG脚本 VLA模型实时推理，动态规划
执行 固定动作序列 根据上下文灵活组合行为
学习 无 访客行为数据→持续优化策略
预测 无 预测访客需求，提前行动

2.2 五大主动式能力

能力1：自主感知与发现

场景 固定响应 SesameX主动式
地面有水 等洒漏触发按钮 M20巡检时视觉+红外主动发现，立即处理
访客在看展品 无响应 Walker看到访客停留→主动走过去讲解
咖啡杯空了 等访客按钮 咖啡机视觉识别空杯→主动询问续杯
访客迷路 无响应 Walker观察访客行为→主动靠近引导

能力2：动态决策

AI Agent理解全局上下文 → 动态生成最优方案

示例：
  场景：访客A在看芯片展品 + 访客B在拍照 + M20在巡检

  固定响应：按预设流程走，不管实际情况
  主动式Agent决策：
    1. 判断A停留时间长→Walker先去给A讲解
    2. 预判A讲解完会想喝咖啡→咖啡机预热
    3. B在拍照→不打扰，等B拍完再引导
    4. M20继续巡检→不打扰访客体验

能力3：个性化交互

访客画像生成：
  摄像头识别 → 年龄/性别/穿着/表情/停留位置 → 实时画像

个性化行为：
  年轻技术男 → Walker用技术术语讲解芯片架构
  政府领导 → Walker用产业价值角度讲解
  小朋友 → Walker蹲下打招呼，用简单语言
  外宾 → Walker切换英语，调整内容权重

能力4：行为预测

预测 依据 主动行为
访客想喝咖啡 在咖啡站附近停留>30秒 咖啡机主动启动
访客对采摘感兴趣 在世界树前停留>15秒 机械臂准备启动
访客要离开 向出口方向移动 Walker主动送别
人流即将拥挤 入口人流计数>阈值 Walker引导分流
Walker电量不足 电量<25%且有访客 完成当前导览再回充

3.2 主动式迎宾全流程（详细）

传统流程：
  RFID签到 → Walker说"欢迎光临" → 按路线导览

主动式流程：
  1. 访客进入展厅入口（无RFID，纯视觉感知）
  2. Walker S3摄像头识别：
     - 人数：1人/2人/团体
     - 特征：年龄/性别/穿着/表情
     - 行为：走快/走慢/东张西望
  3. SesameX Liora VLA推理（<5ms）：
     - 判断：这是VIP？普通访客？技术型？商务型？
     - 生成个性化问候语
  4. Walker主动行为：
     - VIP（西装+证件）："张总您好，我是Walker S3，今天由我为您导览"
     - 技术型（格子衫+背包）："欢迎！您对哪方面的技术最感兴趣？我可以重点讲解"
     - 小朋友：蹲下"你好呀！想不想看机器狗翻跟头？"
     - 团体：自动规划分流路线，避免拥挤
  5. 持续观察访客反应：
     - 访客走快→Walker加快节奏，少讲多看
     - 访客停留→Walker详细讲解当前展品
     - 访客提问→实时语音理解→SesameX推理→回答

3.3 主动式洒漏自愈（无需人为触发）

传统流程：
  人为洒水 → 按触发按钮 → M20检测 → 中枢派单 → UP清洁

主动式流程：
  1. M20 Pro每30分钟巡检（持续感知，不等触发）
  2. 红外热成像+视觉检测地面异常：
     - SesameX Aura实时分析：温差>2°C + 面积>100cm²
     - 连续3帧确认（排除误报）
  3. M20自主决策：
     - 判断严重程度（小水渍/大面积/危险液体）
     - 小水渍→M20自己绕开，通知UP
     - 大面积→M20封锁区域，紧急通知UP+工作人员
  4. 中枢AI Agent编排：
     - 根据当前展厅人流→选择最优清洁路径
     - 如果有VIP导览正在进行→先引导VIP绕行
  5. UP清洁完成→拍照确认→M20复核→闭环

4.3 SesameX在主动式智能中的价值

端侧实时推理（<5ms延迟）：
  - VLA模型：视觉→语言→动作 三通道并行推理
  - 世界模型：预测环境变化，提前规划行动
  - 多模态融合：视觉+听觉+触觉+环境感知

脱离云端也能主动思考：
  - Walker S3断网时仍能自主迎宾/避障/个性化交互
  - M20 Pro断网时仍能巡检/检测异常/自主决策
  - 这就是"SesameX Inside"的核心价值——端侧智能不依赖云端
