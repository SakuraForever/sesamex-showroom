第二章 · 中枢调度系统设计

2.1 任务编排引擎

采用DAG（有向无环图）工作流引擎，每个联动场景定义为一个DAG：


示例：拍照接力DAG

[访客触发] → [Walker拍照] → [人像数据传输] → [咖啡机识别]
                                                      ↓
                                              [咖啡机制作拉花] → [取餐屏更新] → [完成]

| 组件 | 说明 |

|------|------| | 任务定义 | 每个联动场景预先编程为JSON/YAML格式的工作流 |

| 触发器 | 传感器事件 / 手动按钮 / 定时器 / VIP签到 | | 任务节点 | 每个节点=一个机器人的一个动作（含超时/重试） |

| 条件分支 | 支持if-else（如：咖啡豆不足时切换为备用方案） | | 并行执行 | 多个无依赖节点可并行（如：Walker拍照+M20巡检同时进行） |

2.2 路径规划与冲突避免

| 机制 | 说明 |

|------|------| | 时间窗预约 | 机器人通过狭窄通道前向中控申请时间窗 |

| 优先级 | VIP导览（高）> 安全事件响应（高）> 任务执行（中）> 返回充电（低） | | 静态禁区 | 展品区/访客休息区/充电桩周围设为禁区 |

| 动态避让 | 检测到其他机器人3m内时减速，1.5m内停止 | | 行人避让 | 激光雷达+深度相机检测行人，0.5m内急停 |

2.2.1 并发场景调度规则

| 并发场景 | 规则 |

| --- | --- | | VIP导览进行中，普通访客触发拍照接力 | Walker维持VIP导览，拍照接力进入队列，Walker完成当前段落后执行（访客等待不超过5分钟，否则提示改日） |

| 两组VIP同时签到 | 第一组由Walker接待，第二组由工作人员陪同并播放自动导览视频；Walker完成第一组送别后立即切换 | | M20正在越障表演，洒漏事件触发 | 洒漏为安全事件，M20立即中止表演，移动至洒漏位置检测上报；越障表演在完成清洁后重新触发 |

| Walker电量低于25%，VIP导览中 | 导览完成后不执行其他任务，直接返回充电；中控提示工作人员准备接替 | | 多台机器人同时故障（超过2台） | 全局切换降级模式，暂停所有自动场景，仅保留静态展示，通知运营人员 |

场景2 · 拍照接力→专属咖啡

触发条件

普通访客在Walker展位前触发拍照（触摸屏/语音"拍照"）

工程细节

| 步骤 | 硬件动作 | 数据流 | 时长 |

|------|----------|--------|------| | 1.人像采集 | Walker S3胸部摄像头拍照 | 照片→中控→咖啡机 | 3s |

| 2.人脸识别 | 中控服务器运行人脸检测 | 特征点→拉花图案生成 | 2s | | 3.图案传输 | 咖啡机接收拉花图案 | 图案→Liora芯片 | 1s |

| 4.制作咖啡 | 咖啡机萃取+拉花 | 全自动 | 90s | | 5.取餐提醒 | 取餐屏+语音播报 | 人脸ID→取餐匹配 | 5s |

拉花图案生成方案

  • 中控运行轻量级StyleGAN模型（边缘推理）

  • 输入：人脸照片 → 输出：简化的线条画（适合咖啡拉花）

  • 生成时间：<2秒（RTX A6000边缘推理）

  • 图案传输：HTTP API推送到咖啡机

隐私保护

  • 原始人脸照片仅在本地内存处理，特征提取完成后立即销毁，不落盘

  • 人脸特征向量保留至访客取餐完成（用于凭脸匹配），取餐后自动清除

  • 拉花图案在制作完成后30分钟自动清除

  • 人脸识别仅做局部特征提取，不关联身份，不上传云端

| 成功率目标 | >90%（随机位置+恒定光照+颜色分割） |

5.4 抓取流程（总约10秒）


步骤1：供果（自动）
  - 链条提升机将苹果送到顶部，偏转片随机弹向6槽之一
  - 苹果落入V型槽，重力固定

步骤2：视觉扫描（0.5s）
  - D435i扫描6个槽位
  - Aura芯片HSV分割找到苹果，输出6DoF位姿
  - LED大屏实时显示识别过程

步骤3：运动规划（0.3s）
  - Aura芯片运行OMPL/RRT路径规划
  - 规划至苹果正上方，避开槽位边缘

步骤4：接近吸附（2s）
  - 机械臂移动到苹果上方，速度限制0.1m/s
  - 真空泵启动，吸盘接触苹果表面并吸附

步骤5：提起（1s）
  - 机械臂向上提起，苹果脱离V型槽（重力脱离，无需旋转）
  - 负压确认吸附成功

步骤6：递送（3s）
  - 移动到取物台上方，释放真空
  - 苹果沿15°斜面滑落至取物口

步骤7：复位供果（2s）
  - 机械臂回到扫描待机位
  - 供果管自动释放下一个苹果（随机槽位）
  - LED显示"请取走您的水果"
